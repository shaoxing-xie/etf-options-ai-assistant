# 工具速查表

按分类列出所有 OpenClaw 注册工具，便于快速查找。详细参数与返回值见 [工具参考手册](工具参考手册.md) 或 `config/tools_manifest.yaml`。

## 数据采集

| 工具 ID | 标签 | 说明 |
|---------|------|------|
| tool_fetch_index_data | 获取指数数据（统一入口） | merged：`data_type`=realtime/historical/minute/opening/global_spot |
| tool_fetch_etf_data | 获取ETF数据（统一入口） | merged：`data_type`=realtime/historical/minute |
| tool_fetch_option_data | 获取期权数据（统一入口） | merged：`data_type`=realtime/greeks/minute |
| tool_read_market_data | 从缓存读市场数据（统一入口） | merged：index/etf/option 多类型缓存 |
| tool_fetch_index_realtime | 获取指数实时数据 | 主要指数实时行情，盘前/盘中快扫 |
| tool_fetch_index_historical | 获取指数历史数据 | 指数日/周/月 K 线 |
| tool_fetch_index_minute | 获取指数分钟数据 | 指数分钟 K 线 |
| tool_fetch_index_opening | 获取指数开盘数据 | 9:28 集合竞价 |
| tool_fetch_global_index_spot | 获取全球指数实时数据 | 道琼斯、纳斯达克、标普等 |
| tool_fetch_etf_realtime | 获取ETF实时数据 | ETF 当前价（mootdx/TDX -> 同花顺 -> 新浪分类 -> 新浪1min兜底；含短缓存） |
| tool_fetch_etf_historical | 获取ETF历史数据 | ETF 日/周/月 K 线 |
| tool_fetch_etf_minute | 获取ETF分钟数据 | ETF 分钟 K 线 |
| tool_fetch_etf_iopv_snapshot | 获取 ETF IOPV/折价率 | 东财 ETF 列表（IOPV、折价率等） |
| tool_fetch_option_realtime | 获取期权实时数据 | 期权实时行情 |
| tool_fetch_option_greeks | 获取期权Greeks数据 | Delta、Gamma、Theta、Vega 等 |
| tool_fetch_option_minute | 获取期权分钟数据 | 期权分钟 K 线 |
| tool_fetch_a50_data | 获取A50期指数据 | 富时 A50 期指实时/历史 |
| tool_fetch_sector_data | 获取板块轮动数据 | 行业/概念板块（东财优先） |
| tool_get_option_contracts | 获取期权合约列表 | 按标的查合约列表 |
| tool_check_trading_status | 判断交易时间状态 | 返回 `allows_intraday_continuous_wording`、`quote_narration_rule_cn`；写 A 股「盘中/今开」前必调 |
| tool_fetch_stock_historical | 获取个股历史K线 | A 股个股日/周/月 K 线 |
| tool_fetch_stock_minute | 获取个股分钟数据 | A 股个股分钟 K 线 |
| tool_fetch_stock_realtime | 获取个股实时数据 | A 股个股实时行情 |
| tool_fetch_stock_financials | 获取A股财务指标 | PE/PB/ROE/股息率等 |
| tool_fetch_limit_up_stocks | 获取涨停股列表 | 涨停回马枪数据源 |
| tool_sector_heat_score | 板块热度评分 | 涨停回马枪板块热度 |
| tool_write_limit_up_with_sector | 涨停+板块日报写入 | 写入 limit_up_research 目录 |
| tool_limit_up_daily_flow | 涨停每日流程 | 涨停数据+报告+可选飞书 |
| tool_dragon_tiger_list | 龙虎榜 | 龙虎榜数据 |
| tool_capital_flow | 资金流向 | 个股主力/散户资金流向 |
| tool_fetch_northbound_flow | 北向资金流向 | 北向资金数据 |

## 分析

| 工具 ID | 标签 | 说明 |
|---------|------|------|
| tool_calculate_technical_indicators_unified | 统一技术指标入口 | 迁移期推荐入口；与旧工具参数兼容（`engine` 等兼容参数会被忽略），内部按运行时路由计算并返回兼容结构；配置：合并后配置 → `technical_indicators`（域文件：`config/domains/analytics.yaml`）。 |
| tool_analyze_after_close | 盘后趋势分析 | 15:30；`data` 含 **`report_meta`**、**`daily_report_overlay`**（北向/全球/关键位/板块，可选 ADX）；配置 **`trend_analysis_plugin`**；落盘 `data/trend_analysis/after_close/` |
| tool_analyze_before_open | 开盘前趋势分析 | 9:15；`report_meta`；**A50/HXC 仅本模式**；未传盘后结果时优先读 **`after_close` 落盘**，否则重算；返回 **`after_close_basis`**（`disk`/`computed`/`passed`）；`before_open/` |
| tool_analyze_opening_market | 开盘行情分析 | 9:28；`report_meta`；简版开盘情绪等见 **`trend_analysis_plugin.fallback`**；独立 **`opening/`**（`system.data_storage.trend_analysis.opening_dir`）；叙事见 Skill **`ota_trend_analysis_brief`** |
| tool_predict_volatility | 波动率预测 | GARCH/ARIMA 等；侧重 **日内剩余交易时间** 尺度，勿与全日日频工具混用 |
| tool_calculate_historical_volatility | 单窗口历史波动率（年化%） | 单一 `lookback_days`；`fetch_index_daily_em`；多窗口/锥/IV 用下行快照工具 |
| tool_underlying_historical_snapshot | 标的历史波动快照 | 多标的、`hv_by_window`、可选 `vol_cone`、可选 SSE ETF 近月 IV；合并后配置 → `historical_snapshot`（域文件：`config/domains/analytics.yaml`）；runner 别名 `tool_historical_snapshot` |
| tool_historical_snapshot | 同上（别名） | 与 `tool_underlying_historical_snapshot` 等价 |
| tool_generate_option_trading_signals | 生成期权交易信号 | 主名；合并后配置 → `signal_generation`（域文件：`config/domains/signals.yaml`） |
| tool_generate_signals | 生成交易信号（别名） | 与期权主工具等价 |
| tool_generate_etf_trading_signals | 生成 ETF 交易信号 | `signal_generation.etf` |
| tool_generate_stock_trading_signals | 生成 A 股监控信号 | `signal_generation.stock`（非投资建议） |
| tool_strategy_engine | 策略引擎与信号融合 | 多路 Rule 聚合 + Fusion；`summary`/`fused_by_symbol`/多 ETF 逗号参数；综合信号场景优先 |
| tool_assess_risk | 单标的风险评估 | ETF/指数/A 股；`asset_type`+缓存/拉日线；HV=`realized_vol_windows`（%%）；凯利简化；合并后配置 → `risk_assessment`（域文件：`config/domains/risk_quality.yaml`） |
| tool_predict_intraday_range | 预测日内波动区间（含收敛与记录） | 日内价格区间预测（`range_pct` 夹紧、`confidence` 上限等；并落库用于回填/评估） |
| tool_predict_daily_volatility_range | 日频波动区间预测（全日） | **完整交易日** horizon：日 K 多窗口 HV（默认 5/22/63）+ ATR(14) 融合；连续竞价内可有界分钟纠偏；与 `tool_predict_volatility` 的「剩余时段」口径不同 |
| tool_check_etf_index_consistency | 检查ETF与指数趋势一致性 | 相关性/一致性得分 |
| tool_generate_trend_following_signal | 生成趋势跟踪信号 | 基于一致性的趋势信号 |
| tool_detect_market_regime | 市场状态识别 | 研究级：trending/range/risk |
| tool_etf_rotation_research | ETF轮动研究 | 研究级：`rotation_config.yaml`+`etf_rotation_core`；`etf_pool` 可空 |
| tool_strategy_research | 策略研究与回放评估 | 研究级：`config/strategy_research.yaml`；IS/OOS/Holdback 切分、务实版 WFE（无 IS 内寻优）、可选交易成本与复杂度惩罚、Holdback 门禁；`report_data.research_disclaimer` |
| tool_get_strategy_research_history | 策略研究运行历史 | 读 `data/backtest_logs/research_runs.jsonl`（路径见配置 `backtest_log.path`），最近 N 条 JSONL |
| tool_screen_equity_factors | A 股多因子选股（震荡模板） | `success`/quality/degraded；夜盘收尾 `tool_finalize_screening_nightly` |
