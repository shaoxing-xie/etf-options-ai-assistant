# 工具参考手册（概述）

基于 OpenClaw 的 A股 / ETF 交易助手（含可选期权扩展能力），工具清单以 **config/tools_manifest.yaml** 为单一数据源，插件从 **config/tools_manifest.json**（由 `python scripts/generate_tools_json.py` 生成）加载并注册。

---

## 工具总数与分类

| 分类 | 数量 | 说明 |
|------|------|------|
| 数据采集 | 25 | 指数、ETF、期权、期货、个股、涨停、龙虎榜、资金流向、北向等 |
| 时段与编排 | 4 | A股时段、Copilot、可交易性过滤、事件哨兵 |
| 分析 | 15 | 技术指标、盘前/盘后/开盘分析、波动率、信号、风险评估、趋势、研究（Regime/轮动/策略） |
| 风险控制 | 5 | 仓位与硬锁定、止盈止损 |
| 策略效果跟踪 | 5 | 信号记录、表现、评分、权重 |
| 通知 | 5 | 飞书消息/卡片/信号/日报/风险预警 |
| 数据访问 | 6 | 从缓存读取指数/ETF/期权数据 |
| 回测 | 2 | 涨停回马枪回测、参数敏感性 |
| **合计** | **67** | 全部在 OpenClaw 中注册 |

---

## 使用方式

1. **OpenClaw Agent**：在 Agent 配置中声明要调用的工具  
2. **OpenClaw 工作流**：在工作流中编排工具顺序与参数  
3. **Gateway API**：通过 OpenClaw Gateway 直接调用  

工具名以 `tool_` 前缀开头，参数为 JSON。
