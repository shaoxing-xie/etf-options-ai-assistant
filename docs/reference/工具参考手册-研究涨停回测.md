# 研究 / 选股 / 涨停回马枪 / 回测 工具说明

以下工具为研究级或专题能力，均在 OpenClaw 中注册，可在工作流中直接调用。参数以 `config/tools_manifest.yaml` 为准。

---

## 市场与策略研究

| 工具 | 说明 |
|------|------|
| **tool_detect_market_regime** | 市场状态识别。基于 ETF 日线缓存计算短中期动量、20 日波动、60 日回撤，输出 regime：`trending_up` / `trending_down` / `range` / `high_vol_risk` 及置信度。 |
| **tool_etf_rotation_research** | ETF 轮动研究。从缓存读多只 ETF 日线，计算动量/波动/回撤，输出排名与 Markdown 研究摘要（report_data）。 |
| **tool_strategy_research** | 策略研究与回放。复用策略表现/评分/权重工具，生成研究报告与权重调整建议（不直接改盘，由工作流/人工决定）。 |

---

## 选股与财务

| 工具 | 说明 |
|------|------|
| **tool_fetch_stock_financials** | 批量获取 A 股财务指标（PE_TTM、PB、ROE、股息率等），供量化选股或单独查询。 |
| **tool_quantitative_screening** | 量化选股。对候选标的做动量/波动率/流动性/估值因子打分，输出排名与 top_picks。 |

---

## 涨停回马枪数据与流程

| 工具 | 说明 |
|------|------|
| **tool_fetch_limit_up_stocks** | 获取指定日或区间的涨停股列表（可排除 ST、尾盘涨停）。 |
| **tool_sector_heat_score** | 板块热度评分。 |
| **tool_write_limit_up_with_sector** | 拉取当日涨停、板块热度与龙头，写入 `data/limit_up_research/`。 |
| **tool_limit_up_daily_flow** | 涨停每日流程：拉取涨停、板块热度、写 JSON 与报告、可选发飞书。 |
| **tool_dragon_tiger_list** | 龙虎榜数据。 |
| **tool_capital_flow** | 个股主力/散户资金流向。 |
| **tool_fetch_northbound_flow** | 北向资金流向。 |

---

## 个股行情（补充数据采集）

| 工具 | 说明 |
|------|------|
| **tool_fetch_stock_historical** | A 股个股历史 K 线（日/周/月）。 |
| **tool_fetch_stock_minute** | A 股个股分钟 K 线。 |
| **tool_fetch_stock_realtime** | A 股个股实时行情。 |

---

## 回测

| 工具 | 说明 |
|------|------|
| **tool_backtest_limit_up_pullback** | 涨停回马枪策略回测。可配置起止日期、数据目录、策略列表、板块热度阈值、持仓天数、止损/目标收益等。 |
| **tool_backtest_limit_up_sensitivity** | 涨停回测参数敏感性。对关键参数做网格搜索，返回推荐参数。 |

---

详细参数见 **config/tools_manifest.yaml**；速查表见 [工具参考手册-速查.md](工具参考手册-速查.md)。
