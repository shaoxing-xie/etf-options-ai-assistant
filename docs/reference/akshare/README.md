# AKShare 接口说明（本地镜像）

本目录存放 **AKShare** 官方文档的本地副本，用于离线查阅各类数据接口（函数名、参数与示例）。  
内容来自上游项目 [AKShare](https://github.com/akfamily/akshare)，**以官方文档为准**；若接口变更，请对照 PyPI / GitHub 最新说明。

| 文档 | 说明 |
|------|------|
| `AKShare-股票数据.md` | A 股相关接口 |
| `AKShare 指数数据.md` | 指数相关接口 |
| `AKShare 公募基金数据.md` | 公募基金相关接口 |
| `AKShare 期货数据.md` | 期货相关接口 |
| `AKShare 期权数据.md` | 期权相关接口 |

与项目内「工具」定义（`config/tools_manifest.yaml`、`工具参考手册.md`）的关系：本目录描述的是 **底层数据源库** 的接口；项目工具是对这些能力等的封装与编排，二者用途不同。
