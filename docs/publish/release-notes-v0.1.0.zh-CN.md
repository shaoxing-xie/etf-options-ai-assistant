# v0.1.0 发布说明（中文）

发布日期：2026-03-21  
版本标签：`v0.1.0`

## 版本定位

`v0.1.0` 是项目首个公开基线版本，目标是提供可复现、可部署、可协作的 A股 / ETF 交易助手工程基础。  
默认主线为股票与 ETF 场景，期权能力作为可选扩展保留。

## 主要内容

- 建立 OpenClaw 集成基线：多 Agent 协作、工作流调度、工具注册与运行链路打通。
- 提供核心能力闭环：数据采集、分析研判、风控校验、通知触达、运行留痕。
- 完成开源发布基础文件：`README`、`LICENSE`、`SECURITY`、`CONTRIBUTING`、`CHANGELOG`、`.env.example`。
- 建立发布安全闸门：JSON 语法校验、发布安全扫描、CI 工作流（release-gate）。
- 完成文档主线重构：发布与部署文档聚焦 `docs/publish/*`，历史资料进入 `docs/archive/*` 与 `docs/legacy/*`。

## 适用人群

- 希望在 OpenClaw 上构建 A股 / ETF 自动化分析与执行辅助流程的用户。
- 需要可二次开发的交易助手基础框架的开发者与运维人员。

## 已知说明

- 本项目用于研究与工程实践，不构成投资建议。
- 仓库名 `etf-options-ai-assistant` 为兼容保留，不代表当前业务主线（当前主线为 A股 / ETF）。

## 下一步计划（Roadmap）

- `v0.2.x`：策略模块化增强、盘中风险模板扩展、发布文档继续收敛。
- `v1.0.0`：生产部署、回滚与运行巡检流程标准化。
